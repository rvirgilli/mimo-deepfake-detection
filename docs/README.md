# Documentation map

Public documentation in this repository is curated for reusable code, setup, audit tooling, and high-level research context.

Raw execution logs, submitted-paper files, reviewer material, and local run artifacts are intentionally not tracked. Keep those under `local_private/` or another ignored path.

Useful entry points:

- `docs/current/README.md` — current curated documentation overview.
- `docs/current/EXTERNAL_DEPENDENCY_SETUP.md` — local dependency setup.
- `docs/current/ASVSPOOF_DATA_LAYOUT.md` — expected ASVspoof local data layout.
- `docs/current/CODECFAKE_OFFICIAL_SPLITS.md` — CodecFake+ split framing.
- `docs/current/WAVE3_REVISED_PLAN_2026_05_31.md` — current diagnostic research plan.

Generated PDFs, figures, and submission packages should be rebuilt or stored locally rather than committed.
